## Installation

You need to have [Node.js][node-url] and [yarn][yarn-url] (faster then npm) pre-installed. You can use npm if you do not have yarn.

### Development setup

```sh
yarn install
yarn start
```

### Build setup

```sh
yarn install
yarn build
```
